# ☕ Coffee Daily - 咖啡沖煮數據化筆記

**Coffee Daily** 是一款專為咖啡愛好者打造的網頁應用程式，將每天的咖啡沖煮過程、參數與感官體驗，轉化為直覺、可追蹤的視覺化數據，並自動同步至個人的雲端資料庫。

## 💡 設計初衷 (Design Purpose)

在鑽研手沖咖啡的過程中，我發現每一次的沖煮都是變數的排列組合（水溫、粉水比、注水段落、研磨度等）。然而，傳統的紙本筆記或普通的文字紀錄軟體，很難直觀地還原「沖煮當下的動態過程」與「風味的立體感」。

因此，我開發了 Coffee Daily。這不僅僅是一個紀錄工具，它更像是一個**個人的咖啡實驗室後台**。設計的核心目標包含：
1. **數據視覺化**：將抽象的注水節奏與溫度變化，轉換為精準的折線圖與雷達圖。
2. **結構化感官體驗**：透過「預期風味」與「驚喜風味」的標籤分類，幫助梳理味覺記憶。
3. **無縫的工作流**：打造極致流暢的前端體驗，並透過自動化工具將所有數據無痛歸檔至 Notion 進行永久保存與回顧。

## ✨ 核心功能 (Key Features)

* 📊 **動態沖煮曲線 (Brewing Curve)**：支援多段式注水紀錄。自動繪製「累積水量」與「水溫變化」的混合折線/面積圖，精準還原每一段的注水力道與手法。
* 🕸️ **風味雷達圖 (Sensory Radar)**：提供 8 個維度（乾香、溼香、風味、尾韻、酸質、口感、甜感、整體）的滑桿評分，並即時生成專屬的雷達圖。
* 🏷️ **智慧風味標籤庫 (Flavor Tagging)**：內建龐大的咖啡風味描述標籤（如：衣索比亞常見的花香、柑橘；哥倫比亞的酒感等），並支援手動擴充。
* ☁️ **全自動雲端歸檔 (Notion Integration)**：一鍵送出後，前端資料會封裝打包，透過 Webhook 完美串接並寫入個人的 Notion Database，建立專屬的咖啡豆與沖煮設備關聯庫。

## 🛠️ 技術架構 (Tech Stack)

* **前端 (Frontend)**: HTML5, CSS3, Vanilla JavaScript
* **資料視覺化 (Data Visualization)**: Chart.js (支援動態更新、斷線渲染與自訂標籤置中)
* **後端與自動化 (Backend & Automation)**: Make.com (Webhook), Notion API
